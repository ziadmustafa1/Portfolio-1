# Ziad Mostafa Portfolio

Professional portfolio for Ziad Mostafa, a Freelance Full-Stack Software Engineer based in Egypt.

Live: https://ziad-frontend.vercel.app

## Tech Stack

- Next.js 15.3.5
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- next-themes
- React Hook Form
- Yup
- Zustand
- Lucide React

## Features

- Recruiter-ready hero, about, experience, skills, projects, and contact sections.
- ATS-aligned positioning for freelance full-stack engineering work since 2022.
- Real project cards for Wexts, Lexniva, Time Slot Manager, Hikaya Stories, Aetheris AI, and countries-cities-ar.
- Honest language level: Arabic native and English basic.
- Clean SEO metadata and Schema.org structured data without fake analytics or verification tokens.
- Dark mode, responsive layout, and reduced-motion-aware Framer Motion usage.
- Contact options through email, WhatsApp, LinkedIn, and GitHub.

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Useful Commands

```bash
npm run build
npm run lint
```

Note: this repository currently uses Next.js 15, where `next lint` may require a custom lint script if the command is removed by the framework version.

## Deployment

The site is designed for Vercel deployment.

1. Push changes to GitHub.
2. Import or connect the repository in Vercel.
3. Build command: `npm run build`.
4. Output is handled by Next.js.
